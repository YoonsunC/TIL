## Web Application Firewall(WAF)

WAF = 응용 프로그램 수준의 보안 적용

:SQL Injection 공격

URL에 값을 이용해서 Database 대규모 값을 넣는 공격 방법

Script 실행으로 권한 탈취


물리적 장치의 WAF = AWS + DDoS 공격도 막아줌

DDoS 공격 : AWS가 기본은 막아줌

DDoS 공격 대체로 처음에는 약하게 하다가 강하게 들어옴

약 : 무료로 막아줌

강 : 약 월 300만원 정도 지불하고 사용
